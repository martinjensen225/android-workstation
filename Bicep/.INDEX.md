# Bicep Index

- [main.bicep](./main.bicep): Subscription-scope entry point for the Azure VM deployment.
- [main.bicepparam](./main.bicepparam): Canonical parameter file used by the GitHub Actions deployment workflows.
- [modules/dev-vm-stack.bicep](./modules/dev-vm-stack.bicep): Resource-group-scoped VM stack with networking, NSG rules, VM deployment, and bootstrap injection.
- [scripts/bootstrap-vm.sh](./scripts/bootstrap-vm.sh): First-boot bootstrap payload injected into the VM through `customData`.
